-> Frontend - Home Appliance Shop

Frontend của website bán đồ gia dụng, được xây dựng bằng HTML, CSS và JavaScript, sử dụng để hiển thị giao diện và gọi API từ backend.

-> Chức năng chính

- Hiển thị danh sách sản phẩm
- Xem chi tiết sản phẩm
- Tìm kiếm và lọc sản phẩm
- Thêm sản phẩm vào giỏ hàng
- Cập nhật và xóa sản phẩm trong giỏ hàng
- Đăng ký, đăng nhập người dùng
- Gửi request đến backend thông qua REST API

-> Công nghệ sử dụng
- HTML5
- CSS3 / Bootstrap hoặc Tailwind (nếu bạn dùng)
- JavaScript (Fetch API / Axios nếu có)
